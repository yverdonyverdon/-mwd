## Getting Started

To get a local copy up and running follow these simple example steps.


### Scripts

1. Clone the repo
   ```sh
   git clone https://github.com/mosab7/portfolio-main
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run locally
   ```sh
   npm run dev
   ```
4. Build production version
   ```sh
   npm run build
   ```
